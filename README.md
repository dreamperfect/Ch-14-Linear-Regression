# Pizza Sales Prediction and Regression Diagnostics Using Python

This project applies simple linear regression techniques to analyze and predict pizza sales based on population demographics. The analysis explores the relationship between a city's population and pizza sales through statistical modeling, visualization, and diagnostic testing.


The project begins by importing sales data from a CSV file using Pandas. An exploratory analysis is performed to understand the dataset and summarize key statistics. A simple linear regression model is then fitted using both SciPy and Statsmodels to estimate the relationship between population size (independent variable) and pizza sales (dependent variable).
